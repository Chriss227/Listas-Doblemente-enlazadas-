# Listas Doblemente Enlazadas

Este repositorio contiene material relacionado con la implementación y estudio de **listas doblemente enlazadas**, una estructura de datos fundamental utilizada para almacenar colecciones de elementos donde cada nodo mantiene referencias tanto a su nodo anterior como a su nodo siguiente.

Las listas doblemente enlazadas permiten:

- Insertar y eliminar elementos de forma eficiente en cualquier posición.
- Recorrer la lista en ambos sentidos (inicio → fin y fin → inicio).
- Administrar dinámicamente memoria según se agregan o eliminan nodos.
